# 💰 Expense Tracker

A simple desktop Expense Tracker application built with **Qt 6, QML, and C++**. This project was created to practice GUI development and learn Qt Quick application development.

## ✨ Features

- Add expense entries
- View expense records
- Clean and simple interface
- Built using Qt Quick and QML
- C++ backend integration
- CMake-based project structure

## 🛠 Technologies Used

- C++
- Qt 6
- QML
- Qt Quick Controls
- Qt Quick Layouts
- CMake

## 🚀 Getting Started

### Prerequisites

- Qt Creator
- Qt 6.x
- CMake
- C++ Compiler

### Build and Run

1. Open the project in Qt Creator.
2. Configure the Qt kit.
3. Build the project.
4. Run the application.
